# Spring Boot CRUD Practice

이 프로젝트는 Spring Boot 기반 백엔드 개발을 혼자 연습하기 위한
개인 학습용 CRUD 프로젝트입니다.

부트캠프 팀 프로젝트 외 개인 시간에,
기초부터 심화까지 단계적으로 백엔드 개발 흐름을 익히는 것을 목표로 합니다.

## 학습 목적

- Spring Boot 기반 CRUD 구조에 익숙해지기
- REST API 기본 개념과 흐름 이해
- JPA를 사용한 데이터베이스 연동 경험
- “요청 → 처리 → 저장 → 응답” 전체 흐름을 직접 구현해보기
- 혼자서 문제를 해결하며 백엔드 사고력 기르기

## 진행 방식

- 이 프로젝트는 하루에 조금씩, 장기적으로 발전시키는 것을 전제로 합니다.
- 기능을 한 번에 완성하기보다,
  기초 → 확장 → 개선의 흐름으로 반복 구현합니다.
- 부트캠프 팀 프로젝트 이후 개인 시간에 진행합니다.
- 이해하지 못한 개념은 기록하고, 다시 구현하며 정리합니다.
- 각 프로젝트마다 README.md에 트러블슈팅을 기록합니다.

## 단계별 학습 범위

### 1단계: CRUD 기본 흐름 이해
- Spring Boot 프로젝트 생성
- 단일 Entity 기반 CRUD 구현
- Controller / Service / Repository 구조 이해
- Postman을 통한 API 요청 테스트
- “요청 → 서비스 처리 → DB 저장 → 응답” 흐름 체득

### 2단계: 데이터베이스 연동과 환경 이해
- MySQL 연동
- Docker를 이용한 DB 환경 구성
- application.yml 설정 이해
- DB 연결 오류 및 트러블슈팅 경험
- JPA 기본 동작 방식 이해

### 3단계: 구조 개선과 코드 분리
- Service 계층 책임 정리
- DTO 도입
- 응답/요청 객체 분리
- 기본적인 예외 처리 구조 학습

### 4단계: CRUD 확장
- 단건 조회 / 수정 / 삭제 구현
- 페이징, 정렬 기초
- 간단한 검색 조건 추가
- 엔티티 확장에 따른 구조 변화 경험

### 5단계: 보안 기초 학습
- Spring Security 기본 개념 이해
- 인증(Authentication) / 인가(Authorization) 흐름 학습
- JWT 기반 로그인 처리 연습
- 토큰 발급 및 검증 구조 이해

### 6단계: 인증 방식 확장 (선택)
- OAuth 개념 학습
- 소셜 로그인 흐름 이해
- Security + JWT + OAuth 구조 비교



## 사용 기술

- Java
- Spring Boot
- Spring Data JPA
- MySQL (Docker)
- Gradle
